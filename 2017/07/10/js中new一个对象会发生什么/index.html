js中new一个对象会发生什么

发表于 2017-07-10 | 分类于 JavaScript

js中可以使用new构造函数的方法创建一个新对象，使用这种方式调用构造函数实际上会经历以下步骤：

1、创建一个新对象
2、将构造函数的作用域赋给新对象（this）
3、执行构造函数中的代码（为新对象添加属性）
4、返回该对象。
因此对于一个构造函数即使它的内部语句最后并没有return，也会默认有return this；语句。
